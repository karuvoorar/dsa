# Linked List

## What is it?

1. A non-contagious array, dynamically varying in size.
2. Starts with an element called `Head`, link to subsequent elements is via `Next Node` of the LL.
3. End of LL is denoted by the `Next Node` pointing to `null` which is called `tail`

## Usage
1. Stack/Queue
2. Browser
3. Switching Apps in mac via cmd + tab
