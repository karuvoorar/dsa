# Greedy

Activity / Interval Selection

- Idea: Pick the task that finishes earliest.
- Key sort: By end time.
- Used when: Max number of non-overlapping intervals.
- Examples: Meeting rooms, activity selection.

Interval Merging / Covering

- Idea: Sort intervals, merge or cover greedily.
- Key sort: By start (or end for covering).
- Used when: Overlaps, coverage, minimum arrows.
- Examples: Merge intervals, minimum arrows to burst balloons.
